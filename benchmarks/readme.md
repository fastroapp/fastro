# Benchmarks

__Machine__: MacBook Air, 1.8 GHz Dual-Core Intel Core i5, 8 GB 1600 MHz DDR3

__Node.js__: v14.3.0

__Deno__: v1.0.1

## Method

Start all server:
```
npm start
```
Compare:
```
npm run compare
```

Compile the benchmark results and write them to readme:
```
npm run compile
```

Shutdown server:
```
npm run kill
```
